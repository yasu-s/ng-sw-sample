# Overview

Angular + ServiceWorker sample.


# System requirements

* Node.js - 10.x
* Yarn - 1.12.x

# Used library

* TypeScript - 3.5.x
* Angular - 8.2.x
* @angular/pwa - 0.803

# Usage

## 1. Download Sample

```
git clone git@github.com:yasu-s/ng-sw-sample.git
```

## 2. Installing packages  

```
cd ng-sw-sample
yarn
```

## 3. Build the sample  

```
yarn build:prod
```

## 4. Launch sample application  

```
yarn serve
```

## 5. Execution result

http://localhost:8080/
